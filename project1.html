Facebook 粉絲團抽獎工具

因為外部沒有好用的 Facebook 抽獎程式再加上FB目前後台讀取十分緩慢，所以我自己寫了一個簡易的工具。
這個工具可以幫助 Facebook 粉絲團管理員從指定貼文中抽出留言者，適合用於留言抽獎活動。
由於 Facebook API 限制，留言者姓名無法直接顯示，只能整理出抽獎人的網址，點進去才能看到得獎者是誰。

使用流程

  1. 輸入貼文關鍵字，搜尋包含該關鍵字的貼文
  2. 選擇關鍵字（常用留言抽獎方式）
  3. 設定抽獎的開始與結束時間
  4. 輸入要抽出的人數，點選「開始抽獎」
  5. 下方會顯示抽獎結果，包含得獎者的 Facebook 網址

優缺點

  • 優點：至少能整理出抽獎人的網址，方便確認得獎者身分，省去人工比對步驟
  • 缺點：無法直接取得留言者姓名（Facebook API 限制）
