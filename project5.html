Discord Bot + 後台管理系統

本專案結合 Discord Bot（discord.js v14）與 Express 後台 API，並提供分頁式 Dashboard 前端，方便管理與互動。

主要功能

  • 分頁式 Dashboard 前端，支援伺服器/頻道/Thread 分組選擇
  • 抽獎、投票、活動紀錄、數據分析等分頁式管理
  • 檔案搜尋、排序、輸出、刪除，活動紀錄 CSV 解析與分布圖表
  • 互動式 UI（Chart.js 圖表、分頁切換、檔案管理、進度條、匯出 JSON）
  • 訊息與數據分析（活躍時段、成員分布、最佳公告時段建議等）
  • API 與權限安全，頻道/Thread 權限檢查，支援多伺服器管理
  • Bot 基本功能：登入提示、!ping 指令、可擴充自訂指令

使用流程

  1. 選擇伺服器、頻道或 Thread 進行管理
  2. 分頁切換抽獎、投票、活動紀錄、數據分析等功能
  3. 檔案管理、活動紀錄分析、圖表化、匯出資料

優缺點

  • 優點：分頁式管理、互動式 UI、數據分析、權限安全、支援多伺服器
  • 缺點：需有 Discord Bot 權限、伺服器規模大時資料量多需優化
